== Description ==

Use the Burd Delivery Shipping plugin for WooCommerce to offer your customers same- and flex-day delivery.

Features:
- Same day delivery.
- The user can choose when they want the package delivered when Flex-option is enabled.
- Dynamic checkout, if same-day delivery is not available Burd Delivery will automatically display the next possible delivery day.
- Print labels directly through the Plugin

Above options can easily be setup individually, depending on your wishes and needs.



== Installation ==

1. Install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Woocommerce --> settings --> Shipping --> Zones --> Add Burd Delivery and fill out the settings.
4. Go to Woocommerce --> settings --> Shipping --> Shipping settings --> Fill out the information under "Burd options"
